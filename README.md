# GuessTheWord

A simple and interactive Hangman-style word guessing game built using Python.

## Features

- Random word selection
- Limited lives system
- ASCII Hangman stages
- Letter-by-letter guessing
- Win and lose conditions
- Command-line gameplay
- Beginner-friendly Python project

## Technologies Used

- Python
- random module

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/choudharyabhinavkumarroy-bot/GuessTheWord.git
```

### 2. Open the project folder

```bash
cd GuessTheWord
```

### 3. Run the game

```bash
python main.py
```

## How to Play

- Guess one letter at a time
- Correct guesses reveal letters in the hidden word
- Wrong guesses reduce your lives
- Hangman drawing updates after wrong guesses
- Game ends when:
  - You guess the complete word (Win)
  - Lives become 0 (Lose)

## Game Preview
